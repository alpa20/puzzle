Debug me in local¶

  1. npm install spm -g && npm install spm-doc -g
  2. git clone git@github.com:popomore/puzzle.git
  3. cd puzzle
  4. spm doc watch
  5. open http://127.0.0.1:8000

More detail in http://aralejs.org/docs/develop-components.html
